Command-line argument parser and command router for SharpCrypterCLI

Class Overview

The Parser class handles command-line argument processing and routes commands to appropriate handlers. It supports encryption, decryption, and various utility commands.

Main command processor for SharpCrypterCLI

Parameters

Command-line arguments array

Supported Commands

--help | --version | --encrypt | --decrypt | --launch-gui | --examples

Encryption/Decryption Options

-f | --file       Specify input file
-d | --directory  Process directory
-s | --secret-key Encryption key
-o | --overwrite  Overwrite files
-m | --method     Encryption method (caesar/aes/rsa/des/3des/blowfish)

Example Usage

sharp-c --encrypt -f document.txt -s mykey -m aes -o
sharp-c --decrypt -d ./encrypted/ -s mykey --format .scr
Security Note: Caesar cipher is provided for demonstration purposes only and should not be used for sensitive data.
