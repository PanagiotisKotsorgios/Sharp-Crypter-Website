Namespace: SharpCrypterCLI
Class: Decryptor

XOR-based decryption component for symmetric cipher operations

Class Overview

Provides fundamental XOR-based decryption functionality. Implements symmetric key decryption where the same key is used for both encryption and decryption.

Security Note: XOR encryption provides basic obfuscation but should not be considered secure for sensitive data.

Decrypt Method

public static string Decrypt(string data, string key)

Parameters

data
Encrypted string data to decrypt
key
Decryption key (must match encryption key)

Returns

Decrypted string using XOR cipher

Algorithm

decrypted[i] = (char)(data[i] ^ key[i % keyLength])

XOR Cipher Details

Symmetric Key Operation

Data Key XOR Result
A (65) K (75) 26 → SUB

Characteristics

  • Symmetric encryption/decryption
  • Key repetition if shorter than data
  • Bit-level transformation
  • Lossless conversion
Important: Key security is critical - same key must be used for both encryption and decryption.
