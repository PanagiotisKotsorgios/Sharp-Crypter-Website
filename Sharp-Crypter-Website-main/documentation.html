Main entry point and core application logic for SharpCrypterCLI

Introduction

This file contains the main entry point and core error handling functionality for the SharpCrypterCLI application. It handles command-line argument processing, exception management, and error logging.

Key Features

  • Command-line argument validation and routing
  • Comprehensive error handling with logging
  • User-friendly error messaging
  • Persistent error logging for post-mortem analysis

Main Method

static void Main(string[] args)

Main entry point for the SharpCrypterCLI application

Parameters

args
Command-line arguments passed to the application

Exceptions

  • ArgumentException: Thrown for invalid or missing arguments
  • Exception: Thrown for unhandled exceptions

Example

SharpCrypterCLI --encrypt --file document.txt

Remarks

Typical execution flow:

  1. Argument validation
  2. Command routing via Parser class
  3. Error handling and logging for exceptional cases

LogError Method

private static void LogError(Exception ex)

Logs exception details to a persistent error log file

Parameters

ex
Exception object containing error details

Log Details

  • Full timestamp with millisecond precision
  • Exception type
  • Stack trace
  • Inner exceptions (if present)
