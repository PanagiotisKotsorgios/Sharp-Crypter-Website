FileHandler Class

Namespace: SharpCrypterCLI
Class Type: Static
Version: 0.0.1

Secure file I/O operations handler for encryption/decryption workflows

ReadFile Method

public static string ReadFile(string path)

Parameters

Parameter Type Description Validation
path string Absolute or relative file path File existence check

Returns

UTF-8 encoded file content as string

Exceptions

  • FileNotFoundException: Invalid path
  • IOException: File in use
  • UnauthorizedAccessException: Permission issues
  • PathTooLongException: Path exceeds system limits

Example

string content = FileHandler.ReadFile("document.txt");

WriteFile Method

public static void WriteFile(string path, string content)

Parameters

Parameter Type Description Validation
path string Output file path Directory existence
content string Data to write Null check

Exceptions

  • DirectoryNotFoundException: Invalid path

Example

FileHandler.WriteFile("encrypted.txt", encryptedData);

Exception Handling

Common Error Codes

Error Code Condition Recommended Action
0x80070002 File not found Verify path existence
0x80070005 Access denied Check permissions
0x80070070 Disk full Free storage space
Error Handling Strategy:
  • Use try-catch blocks for file operations
  • Implement retry logic for IOExceptions
  • Validate paths before operations
