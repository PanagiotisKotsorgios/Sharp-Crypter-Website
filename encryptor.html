Encryptor.cs

Namespace: SharpCrypterCLI
Class: Encryptor
Version: 0.0.1

Core encryption/decryption functionality using Caesar cipher

Class Overview

Provides file encryption and decryption capabilities using the Caesar cipher algorithm. Handles text file operations and basic security checks.

Security Note: Caesar cipher is a simple substitution cipher and should not be used for sensitive data protection.

CaesarEncryption Method

public static string CaesarEncryption(string filePath, int shift, string outputFileName, bool overwrite)

Parameters

filePath
Path to source file (UTF-8 text)
shift
Character shift value (1-25)
outputFileName
Optional custom output filename
overwrite
Allow file overwriting (default: false)

Process Flow

  1. Read input file as text
  2. Apply Caesar cipher transformation
  3. Generate output filename if not provided
  4. Validate overwrite permissions
  5. Write encrypted file

Example Usage

Encryptor.CaesarEncryption("document.txt", 3, "secret.enc", true)

CaesarDecryption Method

public static string CaesarDecryption(string filePath, int shift, string outputFileName, bool overwrite)

Parameters

filePath
Path to encrypted file
shift
Character shift value (must match encryption shift)
outputFileName
Optional custom output filename
overwrite
Allow file overwriting

Error Handling

  • FileNotFoundException - Invalid file path
  • UnauthorizedAccessException - Write permissions
  • ArgumentException - Invalid shift value

Example CLI Command

sharp-c --decrypt -f secret.enc -s 3 -o
