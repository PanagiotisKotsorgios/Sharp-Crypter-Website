Privacy Policy

Effective: [2025] ¬∑ Last Updated: [2025]

1. Data Collection

Sharp Crypter is designed with a fundamental privacy principle:

  • We do not collect personal information
  • No usage analytics or tracking
  • Zero data transmission from the application

2. Local Processing

All cryptographic operations occur locally on your device:

  • Encryption/decryption keys never leave your machine
  • Files processed in temporary memory only
  • No cloud storage or transmission of processed data

3. Third-Party Services

The Software interacts with external services only when:

  • Checking for updates (optional, disabled by default)
  • Downloading dependencies during installation
  • Voluntary error reporting (explicit user consent required)

4. Security Measures

While we implement robust security practices:

  • Memory is wiped after operations
  • Sensitive data never persists to disk
  • Regular security audits of codebase

Note: Ultimate security depends on your execution environment

5. User Responsibilities

You agree to:

  • Secure your encryption keys
  • Maintain environment security
  • Assume responsibility for data backups

This policy applies to all versions of Sharp Crypter
